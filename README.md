# freelance_AI_front

Фронтенд для платформы фриланса с AI-ассистентом.

## Технологии

- [Next.js](https://nextjs.org) 15
- React 19
- TypeScript
- Tailwind CSS

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для production

```bash
npm run build
npm start
```
